What is ThinEmulator ?

Team ThinEmulator

Project ThinEmulator is an initiative to enable cloud-based smartphone emulator platform. It enables user to create and launch smartphone emulators on-the-go for various purposes such as testing of smartphone application. These emulators can be remotely accessed from any modern web browser which supports HTML5/JavaScript without installing any third party libraries.

Frequently asked questions

Team ThinEmulator

Why Emulator as a service ?

As mobile platforms are growing at great speed, developers need to test and support as many devices as possible. These devices have heavy configurations and purchasing new hardware in order to support emerging trends can incur huge cost.
ThinEmulator takes that load off from development environment and eliminates the need for heavily configured machines for development and testing environment.


How many versions of Android it can support ?

It can support all the versions of Android. Emulators can be created and launched in just few clicks.


What it is not ?

It is not a virtual mobile to keep your personal applications.
